# Gen AI Startup Final Project Requirements

## Course
Generative AI and Applications

## Project Theme

Imagine that you are running a startup, enterprise, organization, or business.

Your task is to design and develop a Generative AI based chatbot or AI tool that solves a real problem for your users.

The goal of this project is to apply Generative AI concepts to real world business scenarios.

---

# Project Objectives

Students should demonstrate:

1. Understanding of Generative AI concepts
2. Prompt engineering skills
3. Ability to design AI powered solutions
4. Basic application development skills
5. Problem solving and creativity
6. Ethical awareness in AI applications

---

# Project Requirements

Your project must include all of the following:

## 1. Startup or Enterprise Idea

Clearly explain:

- Name of your startup or enterprise
- Industry or domain
- Target users
- Business goal

Example:

"AI Travel Buddy is a travel assistant startup helping tourists in Japan plan affordable and efficient trips."

---

## 2. Problem Statement

Describe:

- What problem are you solving?
- Why is this problem important?
- Who faces this problem?

Example:

"International students often struggle to understand train routes and local transportation in Tokyo."

---

## 3. AI Tool or Chatbot

Develop a working AI application.

Possible examples:

- Customer support chatbot
- AI study assistant
- Resume reviewer
- Interview preparation assistant
- Travel planner
- Restaurant recommendation assistant
- AI health information guide
- Product recommendation system
- AI writing assistant
- AI language learning tutor

---

## 4. Technical Requirements

Students should use:

- Python
- Streamlit
- OpenRouter API or OpenAI API or Gemini API
- Prompt engineering techniques

The application should include:

- User input
- AI generated output
- Clean interface
- Error handling

---

## 5. Prompt Design

Students must explain:

- System prompt used
- Why the prompt was designed this way
- How prompt engineering improved the output

---

## 6. User Interface

The project should have:

- Title
- User input area
- Generate button
- Output section
- Clear layout

Bonus marks for:

- File upload
- Sidebar settings
- Multiple features
- Better UI design

---

## 7. Example Inputs and Outputs

Provide at least:

- Three example prompts
- Three example AI responses

---

## 8. Ethical Considerations

Students must discuss:

- Possible misuse of the tool
- Bias or hallucination risks
- Privacy considerations
- Responsible AI use

---

# GitHub Submission Requirements

Students must create a GitHub repository containing:

1. README.md
2. Python code
3. Screenshots
4. requirements.txt
5. Presentation slides
6. Project explanation

---

# README Structure

The README file should include:

1. Project title
2. Startup idea
3. Problem statement
4. Features
5. Technologies used
6. Installation steps
7. Screenshots
8. Future improvements

---

# Final Presentation

Each team will present:

1. Startup or enterprise idea
2. Problem being solved
3. Demo of application
4. Prompt engineering strategy
5. Technical implementation
6. Ethical considerations
7. Future scope

Presentation time:
5 to 7 minutes
