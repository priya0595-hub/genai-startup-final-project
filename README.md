

# Gen AI Startup Final Project

## Project Theme

You are running a startup or enterprise. Your task is to create a chatbot or Gen AI tool that solves a real problem for your users.

## Project Requirements

Your project must include:

1. Startup or enterprise idea
2. Target users
3. Problem statement
4. Gen AI chatbot or tool
5. Simple Streamlit interface
6. Prompt design
7. Example inputs and outputs
8. Ethical considerations
9. GitHub documentation
10. Final presentation

## Example Project Ideas

1. AI customer support chatbot
2. AI career coach
3. AI study assistant
4. AI travel planner
5. AI restaurant assistant
6. AI HR onboarding chatbot
7. AI product description generator
8. AI document summarizer

## Submission

Submit your GitHub repository link on Moodle.
